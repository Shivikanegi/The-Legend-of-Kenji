Thanks for your interest in contributing to The Legend of Kenji.

Guidelines
- Keep changes small and focused.
- Open an issue to discuss larger UX or layout changes before beginning work.
- Optimize image assets for web (use reasonable compression and reasonable dimensions).

How to contribute
1. Fork the repository.
2. Create a branch named `fix/description` or `feature/description`.
3. Commit your changes with clear messages.
4. Open a pull request describing what you changed and why.

Code style
- This is a small static prototype; prefer plain, readable HTML/CSS/JS.
